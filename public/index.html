Launch display ↗
1

Connect to the same network

Make sure this computer and your smart TV are using the same Wi‑Fi or wired network.

2

Find this computer’s address

When the app starts, it prints a “Network” address. It looks like http://192.168.1.20:4173/display.html.

3

Open it on the TV

Enter that address in the television’s web browser. Choose full-screen mode and leave it open.

QUICK START

Help guide

From empty board to brilliant beer list in a few minutes.

01

Build your list

Open Screens, name each screen and add sections such as Cask or Keg. Add as many beers as you need.

02

Add pump clips

Turn on “Show pump clip” for a beer, then paste a direct image address. Whats On Tap scales it neatly without stretching.

03

Style the board

Use Appearance to choose your background, text, accent colour, type size and rotation speed.

04

Show it on TV

Start the app on a computer, keep it running, and open the network display address in the TV browser.

Good to know

Changes are stored automatically in this browser. Refresh the TV display after editing to pick up the latest list.
